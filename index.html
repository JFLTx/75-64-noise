I-75 / I-64 Noise Study - Newtown Pike to Western Split

Methodology

Traffic noise data analyzed for the I-75 / I-64 Widening between Newtown Pike and the Western Split using TNM 2.5. Data was prepared within ArcGIS Pro for use in this analysis.

Receptors were grouped based on location. If a receptor occurs within the front row (closest to the project), it has a suffix "-F". If a receptor occurs at an apartment and is behind the apartment (furthest from the project) it has the suffix "-R". Those groups with their receptor naming conventions are as follows:

  • Residential Properties (Rxx)
  • Legacy Trail (LTxx)
  • Legacy Business Park (Undeveloped) (LBPxx)
  • Kearney Ridge Apartments (Existing) (KRxx-EX)
  • Kearney Ridge Apartments (Under Construction) (KRxx-CON)

Noise values in dB(A). Receptor height at a standard 5 ft, except for apartments where each floor was modelled at 17 ft, and 29 ft for second and third floors, respectively.

Barrier costs are defined in KYTC's Noise Abatement Policy, where KYTC assumes an average cost of $32/square foot of barrier wall. Barriers are deemed reasonable so long as Cost Benefit Ratio (CBR) values do not exceed $40,000 per benefited receptor.

Barrier costs are calculated as follows:

C = L × H × $32/ft²

Where C = Cost ($), L = Length of Barrier (ft), and H = Barrier Height (ft).

CBR is defined as follows:

CBR = C B

Where C = Cost ($), and B = Number of Benefited Receptors.
